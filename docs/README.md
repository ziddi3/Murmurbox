# ShineChain Experience - GitHub Pages

This directory contains the files for the ShineChain Experience GitHub Pages website. The website showcases the Murmur Box: Dimensional Tuner project and provides information about the ShineChain mythology.

## Structure

- `index.html` - The main page of the ShineChain Experience website
- `assets/` - Directory containing all assets for the website
  - `css/` - Stylesheets
    - `styles.css` - Main stylesheet
    - `tuner.css` - Styles for the interactive dimensional tuner
  - `js/` - JavaScript files
    - `main.js` - Main JavaScript file for interactivity
  - `images/` - Directory for image assets (currently using external image URLs)

## Development

To modify the website:

1. Edit the HTML, CSS, or JavaScript files as needed
2. Test locally by opening `index.html` in a web browser
3. Commit and push changes to the repository

## GitHub Pages Configuration

This website is automatically deployed to GitHub Pages from the `/docs` directory in the main branch. No additional configuration is needed as long as GitHub Pages is enabled for this repository.
